# shuffle-sort

## Prerequisite
#### 1.Node version > 14 ->v14.15.4

#### 2.NPM version > 7 ->7.11.2

#### 3.Install yarn globally if not installed
```
npm install -g yarn
```
#### 4.Install vue cli tool globally
```
npm install -g @vue/cli
```

## Project setup
```
git clone https://github.com/srikanthi2i/shuffle-sort.git
```
```
cd shuffle-sort
```
```
yarn install
```
